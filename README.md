# Resetknop 🔴🟠

**Een gratis hulpmiddel voor mensen met ADHD in crisis.**

Gemaakt door iemand die begrijpt hoe het voelt — vanuit ROOD.

---

## Wat is dit?

Resetknop is een eenvoudige, offline bruikbare webpagina die je helpt als je vastzit in een ADHD-meltdown of shutdown. Geen app om te installeren. Geen account. Geen advertenties. Gewoon openen en volgen.

Er zijn twee routes:

- 🔥 **ORANJE** — Overbelast, boos, te veel prikkels
- ⛔ **ROOD** — Shutdown, numb, leeg

Elke route bevat stap-voor-stap begeleiding gebaseerd op bewezen technieken: fysieke beweging, koude wateractivatie (amygdala-reset), ademhaling, grounding en schrijven.

---

## Hoe werkt het?

1. Open `index.html` in een browser — of ga naar [resetknop.nl](https://resetknop.nl)
2. Kies je staat (ORANJE of ROOD)
3. Volg de stappen één voor één

Geen scrollgedrag. Eén stap tegelijk. Zo min mogelijk cognitieve belasting.

---

## Technieken in de tool

| Stap | Techniek | Werking |
|------|----------|---------|
| Koud water | Amygdala-reset | Koud water op polsen/gezicht stuurt een "stop"-signaal naar het alarmsysteem in je brein |
| Ademhaling 4-4-6 | Parasympathisch zenuwstelsel activeren | Verlengt de uitademing om je hartslag te verlagen |
| Grounding 5-4-3-2-1 | Sensorische ankering | Brengt aandacht terug naar het hier en nu |
| Intensief bewegen | Adrenaline afvoeren | Verbrandt de stresshormonen die je lichaam heeft aangemaakt |
| Schrijven | Emotionele ontlading | Ongefiltered schrijven vermindert de cognitieve belasting |

## Zelf hosten

Dit is een enkelvoudig HTML-bestand. Er is geen backend, geen database, geen tracking.

### Via Netlify Drop (snelst)
1. Ga naar [netlify.com/drop](https://netlify.com/drop)
2. Sleep `index.html` erin
3. Klaar — je hebt een live link

### Via Netlify + GitHub (aanbevolen)
1. Fork deze repository
2. Verbind je GitHub-account met Netlify
3. Selecteer deze repo → Deploy
4. Koppel je eigen domein (bijv. resetknop.nl)
